## Documentation

It also exports utils to output with colors in your own CLI Tools.

```go
package main

import (
        "fmt"
	chigo "github.com/UltiRequiem/chigo/pkg"
)

func main() {
	fmt.Println(chigo.Colorize("Hello, World!"))
}
```
